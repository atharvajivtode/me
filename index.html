// about_me

I'm a self-driven embedded systems developer based in Nagpur, India, obsessed with making hardware do interesting things. Whether it's flashing firmware onto a microcontroller at 3AM or debugging a custom SPI display driver, I'm in my element.

My work spans ESP32, ESP8266, Raspberry Pi Pico, and Raspberry Pi Zero platforms — often pushing them well past their default use cases. I enjoy reverse-engineering protocols, building custom HID peripherals, and wiring up automation systems that actually make life easier.

When I'm not deep in hardware, I'm writing Python tooling, scripting automation workflows on Windows, or exploring the layers of systems software that most people never touch.

Hitachi AC IR Controller
Reverse-engineered a custom Hitachi VE-68A IR remote protocol on ESP32 — symmetric 3450µs header, 13-byte MSB-first payload with checksum. Built a Flask home automation interface with DHT11 temperature readout and a two-button timer UI.
ESP32 IRremoteESP8266 Flask GPIO 13 IR LED DHT11
COMPLETE
SimHub Racing Dashboard — AIM MXS
Raspberry Pi Zero 2W ILI9486 SPI racing dashboard receiving live telemetry from SimHub. Iteratively solved rendering via direct framebuffer writes (RGB565 → /dev/fb1) using Python and Pillow after exhausting pygame, Playwright, and headless Chromium.
Raspberry Pi Zero 2W ILI9486 SPI SimHub Python / Pillow framebuffer
